Project Ideas

My idea for a project is to develop a game using Unity. The game would be designed as a shoot ‘em up game. Shoot ‘em up or Shmup is a subgenre for the typical shooter genre in video games. Some popular games of this game genre are space invaders and galaga, where the player controls a space craft of some sort shooting obstacles and enemies above whilst being restricted to the basic character movement (up, down, left and right). The game that I plan to create would be similar to these popular games and will have a science fiction theme. The main goal of the game would be to survive as long as the player can, increasing difficulty as they progress, and achieve higher scores. In order to survive, the player must avoid any enemies and their shots by shooting them first, adding to their total score. The player will be given either a life bar or lives that they should keep watch of because once it is fully depleted, the player will lose.

I believe that this project will be interesting since shoot ‘em up games are very rare in modern video games, making this project unique. There is a huge industry in interactive games and entertainment and there is currently an ongoing growth in the gaming community within Australia as each year passes. This project will bring a unique game genre into the current game industry, giving a fresh genre for new gamers and a nostalgic genre for the older generations. Using a game engine such as Unity, the budget for this project will remain cheap and can be easily programmed after thorough research and studying its tools and capabilities.

The game will consist of a retro theme. I think that this theme would suit very well with he shoot ‘em up genre to keep the basic feel of the game since the controls aren’t difficult to learn. The levels for this game would be somewhat similar to games such as cuphead, easy to learn but difficult to master. It brings a challenge to the game even though the design isn’t too complex. The game would be within the theme of science fiction, since the player will be controlling various space crafts, shooting at other incoming enemy space crafts which can be depicted through different colours or shapes.

In terms gameplay, the player would start off with a standard space craft which can be changed later as the player progresses to unlock other types of ships. These space crafts would need to be balanced, to avoid any abuse of over powered tools and forcing players to pick the “best”. The space crafts would be differed by play style. This can be done by making some faster, deal more damage or have more health. An example would be a tanky space craft would have a significantly more health and deal more damage compared to the standard space craft but can be balanced by making it much slower. The gameplay could also include random power-ups that sometimes appear around the screen. These can give the player buffs that can help them survive much longer such as power-ups that can make them shoot twice as fast, power-ups that can increase their speed or even give them an extra gun to shoot with. Some power-ups could also have the ability to “stack” effects. An example of this would be that if the player picks up two speed power-ups, their added speed would stack twice, making them faster. This might take some time and thinking before it is implemented since it could influence imbalance between the other space crafts once a certain number of power-ups are collected.

Project Ideas

The progression of this game can be made through a levelling system. This can be done by creating levels with certain amounts of experience points required in order to level up. Doing this, the player can be rewarded by unlocking other space crafts. The main goal of the game would be to survive as long as possible. As the player progresses, the enemies would become harder. This can be done by creating more enemies with higher health or do more damage when they shoot, but bosses could also be added to make the game more interesting. This would mean that the player will have to learn their habits to figure out their weaknesses. This will need to be done whilst also avoiding any other enemies and the increasing amount of shots that the bosses and enemies make. Many of these bosses will need many trials in order to fully understand their move set. The idea of the game difficulty is that it is easy to learn but difficult to master. This is because the player is forced to avoid as many shots as possible whilst shooting back in order to survive.

The software that will be required to make this project would be Unity and the operating systems Windows, macOS and Linux. Unity is a game engine developed by Unity Technologies that supports around 27 platforms and can be used to create both two-dimensional and three-dimensional video games. Unity uses the programming language C# and is also able to use JavaScript. The hardware required for this project would be a keyboard which will be used by the player to control their character. The project would also be capable of supporting controllers instead of the keyboard, depending on the player’s preference. This would be added as extra equipment once the project is working.

The necessary skills required to create this project would be general knowledge in programming C#. Since Unity is capable of using the programming language C#, the required skills can be achieved through research and other prior experience with C#. It won’t be difficult to find the required skills, software and hardware required. Since this project will mostly be using C# in Unity, and the game mechanics can easily achieved, the skills required can be found without any problems. The game also wouldn’t require any heavy graphics usages and most people would already own a keyboard, the required hardware is also feasible. The main software used for this project is Unity, which can be downloaded from their website and can be set up for use.

Some problems that can occur when creating this project is my inability to proficiently use C#. Although the project doesn’t require much prior knowledge in using C#, I still might be unable to complete certain tasks. This means that I will require to research any necessary tutorials in order to complete this project. Designing the game could also be difficult as the look of the game can influence many players in either choosing to play the game or avoid it completely. Another issue that I could face when creating this project would be being unable to fully complete the game. With more time and other specialists, creating and finishing the finalised product can be achieved easily.

If the project is successful, the game will bring back an “old favourite” genre into the modern entertainment industry. It can bring a fresh new genre for younger people to experience due to the scarce mounts of shoot ‘em up type games that people currently play. It can also impact older generation gamers with nostalgia due to this being similar to the retro games such as space invaders and galaga.
